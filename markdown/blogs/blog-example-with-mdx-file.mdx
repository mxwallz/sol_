# Getting started with Blog Example with MDX

## What is Blog Example with MDX?

Blog Example with MDX shows you how you can configure a contact form with Tailwind CSS Templates and Bootstrap Templates.

## Why Blog Example with MDX?

### Key Advantages

- **PHP Contact Form**: Configurable with PHP servers.
- **FormBold Integration**: Universal Form API for dynamic forms.
- **Ease of Use**: Simple integration process.

### Best Features

- **PHP Contact Form**: Use with `phpmailer` function for PHP servers.
- **FormBold Integration**: Serverless form backend compatible with all hosting, SSG, and frameworks.

## How to get started

### Steps to Get Started

1. **Using PHP Contact Form**:
   - Replace the email address with your own in `assets->php>contact.php`.
   - Ensure `phpmailer` is up and running smoothly on the server side.

2. **Using FormBold**:
   - Create an account on [FormBold](https://formbold.com/) and create a Form.
   - Integrate the API into your contact form using the provided code.

   - Use the `unique_form_id` from the FormBold dashboard in the `action` attribute.
