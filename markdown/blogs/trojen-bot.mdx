# Getting started with Trojen Bot

### What is Trojen Bot?

Trojen Bot is a powerful AI tool designed to discover hidden cryptocurrency gems. Key benefits include real-time alerts on promising projects and potential investments.

### Why Trojen Bot?

**Key Advantages:**
- **Real-Time Alerts:** Get instant notifications on new and promising crypto projects.
- **Advanced Filtering:** Customize filters to receive alerts that match your investment criteria.
- **Community Insights:** Access community-driven data to make informed decisions.

**Best Features:**
- **Real-Time Alerts:** Receive instant notifications on potential high-reward projects.
- **Advanced Filtering:** Set custom filters to get alerts that match your criteria.
- **Community Insights:** Benefit from community-driven data and discussions.

### How to Get Started

**Step 1:** Join the Telegram group.

**Step 2:** Set up your alert preferences.

**Step 3:** Receive and act on alerts.
